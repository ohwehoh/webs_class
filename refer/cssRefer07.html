웹 사이트에서 이미지를 표현하는 방법

웹 사이트에서 이미지를 표현하는 방법 1. HTML 태그를 통해서 : img 태그 : 의미가 있는 이미지(로고, 메뉴) --> alt(대체 문자)필요! 2. CSS 속성을 통해서 : background 속성 : 의미가 없는 이미지(배경, 색표현) --> ir효과(대체 문자를 CSS 속성으로 표현) 3. 이미지 스프라이트 : background와 ir효과를 통해서 이미지를 표현
