for ...in 반복문

for in 반복문은 배열의 인덱스값이나 객체의 key를 순차적으로 불러올 수 있는 반복문입니다.

for ...of 반복문

for of 반복문은 배열의 값을 불러올 수 있는 반복문 입니다.

for in과 for of의 차이점

for in은 객체의 순환이고 for of는 배열의 순환입니다. 자바스크립트에서는 배열도 객체이기 때문에 for in은 배열의 인덱스 값을 key값으로써 불러옵니다. ::first-line 속성에 선택자를 안주면 document 내의 모든 태그를 받아 각 태그의 첫줄에 스타일 적용을 한다. 화면 크기에 따라 줄바뀜에 따라 오로지 첫번째 줄에만 스타일 적용을 한다. 그래서 이 글자는 검은색으로 나올것!!!!!!!!!!!!!!!!!

자바스크립트의 반복 메소드

기본 동작

배열 내 모든 요소들을 순회하면서 특정 작업을 수행하거나 특정 조건을 만족하는 요소를 가져오는 등 반복적인 작업에 사용한다.

첫번째 인수로 콜백함수를 받는다. 콜백 함수는 일반적으로 다음과 같이 3개의 인수를 사용한다.

function(element, index, array)

forEach()

배열 내 각 요소를 순서대로 처리한다.

forEach() 특징

반환 값이 없다. 원본 배열을 변경시킬 수 있다.

map()

배열 내 각 요소를 순서대로 변환 처리후, 처리 결과를 새 배열로 반환한다.

map() 특징

배열 내 각 요소별로 처리하되, 새로운 결과 배열을 반환한다.

새 배열을 출력하면서 같은 크기를 유지하지만 개별 요소의 내용은 변경 가능
