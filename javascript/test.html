배열 메서드 reduce()

기본형태

배열.reduce((누적값, 현잿값, 인덱스, 요소) => { return 결과 }, 초깃값);

초깃값 설정에 따른 각 매개변수의 출력

let arrNum = [5, 6, 7, 8, 9];
    document.write("

");

매개변수 출력으로 본 reduce 메소드의 흐름

reduce는 배열을 한바퀴 돈다.
초기값 설정을 안하면 인덱스가 1부터 시작한다.
p에는 초깃값 먼저 나오고 그 다음부터 reduce메서드가 리턴한 값,
c에는 인덱스 i의 값
