2. 구구단 출력하기

2.1 2단부터 9단까지 출력하시오!(for문)

{
    for(let i=2; i<=9; i++){
        for(let j=1; j<=9; j++){
            document.write(i + " * " + j + " = " +  i*j + "<br>");
        }
        document.write("=============<br>");
    }
}

2.2 2단부터 9단까지 출력하시오!(while문)

{
    let i = 2, j = 1;

    while(i <= 9){
        while(j <= 9){
            document.write(i + " * " + j + " = " + i*j + "<br>");
            j++;
        }
        j = 1;
        i++;
        document.write("=============<br>");
    }
}
